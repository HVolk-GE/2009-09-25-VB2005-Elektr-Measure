PC Direkt Elektr. Messung

(Step by Step Anweisung)

 

1.)    Starten der Anwendung:

(Main Menu)

 

In diesem Dialog kann man mehr als nur messen oder wiegen.

 

2.)    Zum messen Anmelden:

(messen Anmelden)

 

Start Dialog zum messen und wiegen:

Benutzername; hier muss man seinen „Kurznamen“ (max. 2 Buchstaben) eingeben, danach wird das Kombinationsfeld darunter aktiviert, hier wählt man das auszufüllende Template aus (wird mit Testnummer / Request ID abgespeichert).

Nach dieser Auswahl, kann man selektieren was man messen möchte, Bremsbelag innen / außen oder Bremsscheibe..

 

3.)    Meßzeitpunkt selekieren.

(Meßzeitpunkt)

 

Hier wählt man nun den Meßzeitpunkt aus, wann gemessen wird.

Die Auswahl vom vorigen Dialog wird hier automatisch mit übernommen, nach der Messung gelangt man wieder in diesen Dialog hier her zurück. Hier kann man auch die Auswahl ändern z.B. Bremsscheibe statt Bremsbelag innen / außen.

 

4.) Start Bremsbelag messen / wiegen:

(Belag messen / wiegen)

 

Hier werden die aktuellen Meßpunkte dargestellt. Der aktuelle Meßpunkt wird blau hinterlegt. Mit einem klick auf „Start Messung“ beginnt die Messung der Belagoberfläche.

Mit einem klick auf „Start Wiegen“ wird das Gewicht gelesen.

Die beiden Pfeile am unteren Ende des Dialoges stellen die Einlaufrichtung dar, diese muss bestätigt werden bzw. Ausgewählt. Erst wenn alles Fertig ausgewählt, bzw. gefühlt wurde, kann dieser Dialog geschossen werden und erst beim schließen des Dialoges werden die Daten zu Excel übertragen. Solang der Dialog nicht geschlossen wird, kann man Änderungen vornehmen.

Die sichtbaren Messpunkte, werden aus dem Template gelesen und angezeigt, somit können die Messpunkt, zum hier dargestellten Beispiel unterschiedlich sein.

 

5.)    Start Bremsscheibe messen / wiegen

(Bremsscheibe messen / wiegen)

(Bemsscheibe messen / wiegen)

 

Die Vorgehensweise ist wie unter 4. beschrieben. Auch hier startet man die Messung mit einem klick auf „Start Messung“ und das wiegen mit einem klick auf „Start Wiegen“, jedoch muss man hier die „Disc ID“ und das „Runout“ „einmalig“ eingeben.

Auch hier werden die Daten erst beim schließen des Dialoges an Excel übertragen. Solange bleiben die Werte editierbar.

 

Nach dem schließen der Dialoge  4 und / oder 5 wird der Dialog 3 wieder angezeigt. Zum beenden der Messung -> für ein Template muss man den Dialog 3. schließen und sich neu „Anmelden“.
